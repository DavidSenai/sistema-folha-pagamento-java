# 💼 Sistema de Folha de Pagamento em Java

Sistema desenvolvido em Java com o objetivo de realizar o cadastro de colaboradores e calcular automaticamente a folha de pagamento de acordo com o tipo de funcionário.

---

# 🚀 Funcionalidades

✅ Cadastro de funcionário padrão  
✅ Cadastro de funcionário comissionado  
✅ Cadastro de funcionário de produção  
✅ Cálculo automático de salários  
✅ Geração da folha de pagamento  
✅ Validação de entradas de dados  
✅ Menu interativo no terminal  

---

# 👥 Tipos de Funcionários

## 🔹 Funcionário Padrão
Recebe apenas o salário base.

---

## 🔹 Funcionário Comissionado
Recebe salário base + comissão sobre vendas.

### Fórmula utilizada:

```java
comissao = (vendas * percentual) / 100;
```

---

## 🔹 Funcionário de Produção
Recebe salário base + bônus por produtividade.

### Fórmula utilizada:

```java
bonus = valorPorPeca * quantidadeProduzida;
```
